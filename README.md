A simple Python GUI voice chat application built with Tkinter, gTTS (Google Text-to-Speech), and langdetect.
The app allows users to type messages, automatically detects the language, converts the text into speech, and plays the generated voice message.

✨ Features

🖥️ GUI Interface using Tkinter

🌍 Automatic Language Detection using langdetect

🔊 Text-to-Speech Conversion using gTTS

▶️ Instant Voice Playback after generating audio

🎧 Replay Last Voice Message with one click

📁 Unique MP3 File Generation using timestamps

🔄 Fallback to English if language is unsupported

🛠️ Technologies Used

Python

Tkinter – GUI development

gTTS (Google Text-to-Speech) – speech synthesis

langdetect – language detection

playsound – audio playback

os & time – file handling and timestamp creation
